Introduction

Thank you for purchasing Adelux! Adelux is a modern and elegant HTML5 Bootstrap template specially designed for Padel Clubs & Communities. It is perfectly suitable for any business or service related to padel clubs, sports clubs, court booking, padel academies, athletic centers, racket sports, training clubs, fitness websites, and more.

Requirements

Before editing or customizing Adelux, please make sure you have the following tools installed:

  • Code Editor — Recommended: Visual Studio Code
  • Modern Browser — Chrome, Edge, Firefox, Safari, or Opera
  • Basic HTML/CSS/JS Knowledge — To customize and edit template files

Installation

Follow these steps to start using the Adelux template:

  1. Unzip the downloaded adelux.zip file.
  2. Open the extracted folder in your preferred code editor (we recommend Visual Studio Code).
  3. You can open any .html file directly in your browser to preview the template. However, we highly recommend using the Live Server extension in Visual Studio Code to view the template locally. This ensures that all JavaScript animations, sliders, and interactive elements work properly.
  4. All CSS, JS, and fonts are already linked — no build or compilation is required.

Dependencies

Adelux includes the following libraries and frameworks:

- Bootstrap 5.3.x
- Font Awesome 5x
- jQuery 3.7.1
- Flatpickr Calendar
- Swiper JS Slider

All these dependencies are already included in the assets/js and assets/css folders or loaded via CDN.
